# Quick Start Guide

Welcome to the Happy Coder quick start guide! This guide will help you get up and running with Claude Code in your pocket.

## Installation

1. **Download the mobile app** from your device's app store
2. **Install the NPM package** on your computer:
   ```bash
   npm install -g happy-coder
   ```
3. **Run the CLI tool**:
   ```bash
   happy
   ```
4. **Scan the QR code** with your mobile app

## First Steps

Once you've connected your mobile app to your computer:

1. Open the mobile app
2. Start a new conversation
3. Begin coding with Claude!
