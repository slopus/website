### Static Counter Example

Instead of an interactive counter, here's how you could document a counter component:

```jsx
import { useState } from 'react'

export function Counter({ children }) {
  const [count, setCount] = useState(0)
  return (
    <button onClick={() => setCount(prev => prev + 1)}>
      {children}
      {count}
    </button>
  )
}
```

> Note: Interactive components with hooks need special handling in Nextra for static generation.
